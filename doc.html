Dependency Management

Composer is a tool for PHP developers to manage dependencies. It installs libraries your project needs in a folder called vendor. Composer works on a per-project basis and can also be installed globally if needed.

System Requirements

  • PHP 7.2.5 or higher
  • Command line tools like unzip, tar, gzip
  • Version control tools like Git, Mercurial, Subversion if needed
  • Works on Windows, Linux, or macOS

Installation

Downloading the Composer Executable

  • Locally: Install composer.phar in your project folder
  • Globally: Move composer.phar to a folder in your system PATH

Installation - Windows

Use the Composer-Setup.exe installer or manually download composer.phar and create a .bat file to run it from the command line.

Installation - Docker

You can use Composer via Docker by pulling the composer/composer image and running install commands inside it.

Using Composer

Once Composer is installed, you can declare which libraries your project needs, then run Composer to install or update them.
